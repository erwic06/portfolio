# Building My Personal Website

I recently rebuilt my personal website from scratch. Here's a quick overview of the decisions I made and why.

## Tech Stack

I chose **React** with **Vite** for the build tool and **Tailwind CSS** for styling. This combination gives me:

- Fast development with HMR
- Type safety with TypeScript
- Utility-first CSS that's easy to maintain
- MDX support for writing content in Markdown

## Design Philosophy

I wanted something minimal and clean. The design is inspired by personal sites that prioritize readability and content over flashy animations.

> Good design is as little design as possible. — Dieter Rams

## What's Next

I plan to add more blog posts, refine the design, and maybe add some interactive elements down the road. For now, I'm happy with this simple foundation.
